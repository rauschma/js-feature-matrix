JavaScript feature matrix

This matrix tracks which JavaScript language features are supported by which browsers. There are three ways in which you can contribute:
  • Contribute feature tests: Is there a language feature that once caused you problems by not being supported in a browser?
  • Contribute matrix data: There is a Google spreadsheet for managing the data. Contact me and I’ll give you access.
  • Submit your browser’s results: See section at the end of this page.
GitHub: js-feature-matrix. Contact: axel AT rauschma DOT de or @rauschma.
